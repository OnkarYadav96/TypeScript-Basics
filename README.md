TypeScript is external functionality need to install ==> (npm i -g typescript)
To check typescript version (tsc --version)


To create individual project (npm init -y)  create package.json file
To crate tsconfig file (npx tsc --init)

TypeScript is built on Javascript.Because Broswer knows only Javascript.
After writing code in .ts file ,if we write in terminal "tsc" and enter, Then "dist" folder is created in parallel with src folder
In that dist folder we recieve the same .ts file but that converted our typescript code into Javascript code
